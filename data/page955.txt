6 functional reference architecture of cloud computing 
the layering framework used in the cloud computing reference architecture has four layers, plus a set of 
functions that spans across the layers. the four layers are: 
– user layer; 
– access layer; 
– services layer; 
– resources layer. 
the functions that span layers are called multilayer functions.
