i.12 registration of data storage for federation service 
title registration of data storage for federation service  
description a dsf service customer requests data storage service from a dsf service provider. for the data 
storage registration, dsf service customer logs in through the already registered authentication 
information and registers the cloud storage and on-premises storage to use. the registration 
process is provided by dsf service provider through the graphical user interface (gui).  
dsf service customer registers the service name, storage specification, dat a storage service 
protocol and cloud storage name and cloud storage type through the gui for setting virtual data 
storage and creates the virtual data storage.
